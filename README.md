# CampLus

校园学术生活的操作系统——把学生与课程、知识、选课、信息获取相关的所有需求，从碎片化的 QQ/微信/Piazza/教务系统中抽出来，整合到一个结构化、可靠、不吵闹的平台里。

## 核心系统

| 系统 | 说明 |
|---|---|
| **课程仪表盘** | 每门课的结构化主页，Timeline 为信息核心，学生自治编辑与维护 |
| **个人首页** | 我的课程、截止时间、学业进度——仅课程强相关内容 |
| **知识广场** | Notion 式编辑 + GitHub 式协作，与课程双向链接但不隶属 |
| **选课 × 评课** | 搜索引擎 + 课程档案（成绩/评价数据）+ Schedule Builder |
| **组织与社团** | 公开主页、活动管理、公告发布 |

## 设计原则

[设计原则 →](docs/设计原则.md)

1. **实名身份** — 一切问责和信任的根基
2. **学生自治** — 平台在没有任何教师参与的情况下完整运转
3. **事实优先于情绪** — 围绕事实、时间与责任构建
4. **权力即责任** — 更大权限伴随更可追溯的责任

[完整产品定义 →](docs/产品定义.md)

## 技术栈

- **框架**: React 19 + TypeScript 6
- **构建**: Vite 8
- **路由**: TanStack Router v1
- **样式**: Tailwind CSS v4
- **包管理**: Bun

## 开发

```bash
bun install
bun dev
```
